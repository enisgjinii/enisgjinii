## 🚀 Projects

### 1. [Albatrit-Albulena Wedding Website](https://github.com/enisgjinii/albatrit-albulena)
A beautifully crafted website to celebrate the union of two souls amidst the breathtaking landscapes of the Albatrit and Albulena regions in Kosovo.

- **Tech Stack:** HTML, CSS, JavaScript, PHP, Firebase
- **Features:**
  - Countdown timer
  - Photo gallery
  - Love story section
  - RSVP form with automated email responses

### 2. [Human Resource Management System](https://github.com/enisgjinii/Human-Resource-Management-System)
An HR web application designed to manage employee data efficiently, allowing for the addition, editing, and deletion of employee records.

- **Tech Stack:** PHP, MySQL, jQuery, Bootstrap
- **Features:**
  - Employee CRUD operations
  - Role-based access control
  - Responsive design
  - Data visualization with Chart.js

### 3. [TF-Schwimmbadbau](https://github.com/enisgjinii/tf-schwimmbadbau)
A project focused on construction management, featuring automated email responses and detailed project tracking.

- **Tech Stack:** PHP, MySQL, PHPMailer, Bootstrap
- **Features:**
  - Automated email notifications
  - Project tracking dashboard
  - User authentication
  - Responsive UI

### 4. [Sistemet_e_Bazes_se_te_dhenave](https://github.com/enisgjinii/Sistemet_e_Bazes_se_te_dhenave)
A database management system designed to streamline data operations and improve data integrity.

- **Tech Stack:** Java, MySQL
- **Features:**
  - Database CRUD operations
  - Secure data transactions
  - User-friendly interface
